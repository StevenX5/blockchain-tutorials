# Solidity语言基础教程
本教程由浅入深、循序渐进，对Solidity语言的各个知识点进行了详细的讲解，并结合合约例子进行实践。是一套适用于以太坊智能合约初学者的入门教程。如果您在阅读本教程的过程发现任何问题，请给我们提 [Issues](https://github.com/StevenX5/blockchain-tutorials/issues)。

访问 [这里](https://www.benmo.cc) 查阅更多区块链资料。
